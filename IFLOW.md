# 洛谷算法题解集合

## 项目概述

这是一个C++算法题目解答集合，主要包含洛谷（Luogu）平台的题目解答。项目包含各种难度的算法题目实现，涵盖了动态规划、图论、数论等多个算法领域。

## 项目结构

- **根目录**: 包含所有题目解答文件，每个文件以题目编号命名（如P1002.cpp）
- **gesp-8/**: GESP-8级认证相关的题目和解答
  - `list.md`: GESP-8级题目列表和完成状态
  - `组合数杨辉三角.cpp`: 组合数相关的算法实现

## 文件命名规范

- `Pxxxx.cpp`: 洛谷P系列题目（常规题目）
- `Txxxxxx.cpp`: 洛谷T系列题目（团队题目）
- `Uxxxxxx.cpp`: 洛谷U系列题目（其他题目）
- `test.cpp`: 测试文件

## 编译和运行

### 编译单个文件
```bash
g++ -std=c++17 -O2 P1002.cpp -o P1002
```

### 运行程序
```bash
./P1002
```

### 批量编译（可选）
```bash
# 编译所有.cpp文件
for file in *.cpp; do
    g++ -std=c++17 -O2 "$file" -o "${file%.cpp}"
done
```

## 代码风格和约定

1. **头文件**: 统一使用 `#include <bits/stdc++.h>`
2. **命名空间**: 使用 `using namespace std;`
3. **变量命名**: 主要使用简短的变量名，如 `n, m, i, j` 等
4. **数组大小**: 根据题目需求定义合适的数组大小
5. **注释**: 在文件开头包含题目链接

## 题目分类

根据已分析的文件，项目包含以下类型的题目：

- **动态规划**: P1002（过河卒）、T675692（分组异或和）
- **图论**: P3379（最近公共祖先LCA）
- **数学**: 组合数杨辉三角
- **其他**: 各种基础算法和数据结构题目

## 开发建议

1. 在解答新题目时，建议先在对应的.cpp文件开头添加题目链接注释
2. 对于复杂算法，可以添加适当的注释说明思路
3. 测试时可以使用洛谷平台的样例输入进行验证
4. 建议使用 `-std=c++17 -O2` 编译选项以获得更好的性能

## 注意事项

- 部分题目可能需要处理大数或高精度计算
- 注意数组边界条件，避免越界访问
- 对于递归算法，注意递归深度限制
- 某些题目可能需要优化算法复杂度以通过所有测试点
